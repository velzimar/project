Ecrire un programme qui permet de réaliser les traitements suicants:

Remplir un tableau par n chaines de caractères de taille max 10 formées uniquement par des caractères alphanumérique distincts ( 5 < n < 20 )

Remplir un deuxième tableau d'enregistrement, chaque enregistrement est composé de trois champs:

- Le nombre formé par les chiffres de chaque case

- Le nombre de lettre

- La lettre le plus utilisé (en cas d'égalité en garde la caractère ayant le plus petit code ASCII)

Afficher le contenu du deuxième tableau.
