Ecrire Un programme qui permet de :

- Saisir deux entiers p et q tels que ( 10000 < p < q < 999999 )

- Chercher tous les nombres polydivisibles de l'intervalle [p..q] ainsi que la somme de leurs chiffres, les sauvegarder dans un fichier texte nommé "Polydivisible.txt" de telle sorte que chaque ligne contient un nombre polydivisible suivi du message "est un nombre polydivisible et la somme ds chiffres est" suivi de la somme ds chiffres.

Afficher le contenu du fichier.

Exemple:

345654 est un nombre polydivisible car :

* 34 est multiple de 2

* 345 est multiple de 3

* 3456 est multiple de 4

* 34565 est multiple de 5

* 345654 est multiple de 6
