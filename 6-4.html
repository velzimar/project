Définition: 


Décomposer un entier en facteur premier c'est à dire l'écrire sous la forme de produit de ces facteurs premier.




Traduction en Pascal:


Function Fact_premier ( n : integer) : string ;
var i : integer ;
ch,ch2,ch2 : string;
begin
 i := 2 ;
 ch := '' ;
  repeat 
   if ( n mod i ) = 0 then 
    begin
     str(i,ch2);
     ch := ch + ch2 + '*' ;
     n := n div i
    end;	  
   else
    i := i + 1 ;	 
  until ( n = 1 );
Fact_premier := ch ;
end;
