Présentation :

Un tableau est une structure qui renferme des éléments du meme type.

Par exemple: Pour chercher la meilleur moyenne d'un élève dans une série d'élève, en doit utiliser deux tableaux, un pour définir les noms et autre pour définir les moyennes.

Un enregistrements appeler "record" est une structure qui renferme des éléments du type différent, appeler champ ou attribut.

Déclaration en Pascal :

Type

nom_type = record

champ1 : type 1 ;

champ2 : type 2 ;

...

end;

Var

nom_var : nom_type ;

Manipulation des enregistrements:

Saisir: Consiste à lire le valeur de chaque champ.

analyse: variable.champ = donnée

algorithme: lire (variable.champ)

Pascal: readln (variable.champ);

Affichage: Consiste à afficher ces différents champ.

algorithme: ecrire (variable.champ)

Pascal: write (variable.champ);

Affectation: Consiste à affecter une valeur à chaque champ.

algorithme: variable.champ <= valeur

Pascal: variable.champ := valeur ;
