Suite de Fibonacci:

Soit la suite suivante:

U1=1

U2=1

Un=Un-1+Un-2

0) Def fn Fibonacci_Iterative (n:entier):entier
1) 	Up := 1
	Uap := 1
	Si (n=1) alors
		U := 1
	Sinon si (n=2) alors
		U := 1
	Sinon
		Pour i de 3 à n faire 
			U := Up+Uap
			Uap:=Up
			Up := U
		Fin Pour
	Fin si
2) Fibonacci_Iterative := U
3) Fin Fibonacci_Iterative


0) Def fn Fibonacci_Recursive(n:entier):entier
1)	Si (n=1) alors 
		Fibonacci_Recursive := 1
	Sinon si (n=2) alors
		Fibonacci_Recursive := 1
	Sinon 
		Fibonacci_Recursive := Fibonacci_Recursive(n-1)+Fibonacci_Recursive(n-2)
	Fin si
2)Fin Fibonacci_Recursive

Suite de Robinson:

Le premier terme de la suite de Robinson est posé comme égal à 0. Chaque terme de la suite se construit ensuite en comptant le nombre d'apparitions des différents chiffres de 9 à 0 (dans cet ordre) dans le terme précédent. Si un chiffre n'apparaît pas, il n'est pas pris en compte.

Exemple:

X0=0

X1=10

X2=1110

X3=3110

X4=132110

X5=13123110

X6=23124110

X7=1413223110

X8=1423224110

0) Def fn Robinson (c:caractere ; n:entier):chaine
1) ch := c 
	Pour i de 1 à n faire
		res := ch + ' '
		ch := ' '
		x := 1
		Pour j de 1 à long(ch)-1 faire
			si (res[j]=res[j+1]) alors	
				x := x + 1
			Sinon
				Convch(x,a)
				ch := ch + a + res[j]
			Fin Si
		Fin pour	
	Fin Pour
2) Robinson := Ch
3) Fin Robinson
