Divisibilité par 2:

Un entier est divisible par 2 si son chiffre d'unité est pair.

Divisibilité par 3:

Un entier est divisible par 3 si la somme de ces chiffres est divisible par 3. 

0)Def fn Verif_D3( n: entierlong ): boolean
1)S := 0, Convch(n,ch)
	Pour i de 1 à long(ch) faire
		val(ch[i],x,e)
		S := S + x
	Fin Pour	
2)Verif_D3 := S mod 3 = 0
3)Fin Verif_D3

Divisibilité par 4:

Un entier est divisibe par 4 si le nombre formé de ces deux dèrnier chiffres est divisibe par 4. 

0)Def fn Verif_D4( n: entierlong ): boolean
1)Verif_D4 := n mod 100 mod 4 = 0
2)Fin Verif_D4

Divisibilité par 5:

Un entier est divisible par 5 si le chiffre d'unité est égale à 0 ou 5. 

0)Def fn Verif_D5( n: entierlong ): boolean
1)Verif_D5 := n mod 100 dans [0,5]
2)Fin Verif_D5
